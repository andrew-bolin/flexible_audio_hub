# Hub Connector
Equivalent to (FreeDSP expansion connector)[https://github.com/freeDSP/WIKI-AND-GENERAL-TOPICS/wiki/freeDSP-Guidelines].
16 pin (2x8) header, 2.54mm pitch.

Hub-side

| Function | Pin | Pin | Function |
| ---: | :---: | :---: | --- |
| +12V | 1 | 2 | Gnd |
| I2S in | 3|4 | Gnd |
| I2S out| 5|6 | Gnd |
